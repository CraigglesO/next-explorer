# Next Explorer [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## How to

### Install Dependencies
```sh
# npm
npm install
# yarn
yarn
```

### Run the dev server
```sh
# npm
npm start
# yarn
yarn start
```

### Build the module
```sh
# npm
npm build
# yarn
yarn build
```


## Design

* React - UI Library
* MobX - State Manager
* Flow - Static Type Checker
* Standard - Style Guide
